# FVS — 15 Fichas de Verificação de Serviço Prioritárias para Obras

## Contexto

Este documento reúne um conjunto inicial de **15 FVS — Fichas de Verificação de Serviço** prioritárias para controle de qualidade de obras residenciais e comerciais convencionais.

A estrutura foi pensada para uso em construtoras, reformas, obras novas por empreitada, administração, acompanhamento técnico, fiscalização, planejamento físico-financeiro e padronização de processos executivos.

Cada FVS utiliza campos de verificação com:

- **C = Conforme**
- **NC = Não conforme**
- **NA = Não aplicável**

As FVS devem ser usadas como base para criação de formulários impressos, planilhas, fichas em Word, formulários digitais, sistemas internos ou aplicativos de gestão de qualidade em obras.

---

## Normas e referências gerais

As normas devem sempre ser conferidas em sua versão vigente antes da aplicação final em obra. Como base geral, considerar:

- **ABNT NBR 15575** — Desempenho de edificações habitacionais.
- **ABNT NBR 6118** — Projeto de estruturas de concreto.
- **ABNT NBR 14931:2023** — Execução de estruturas de concreto armado, protendido e com fibras.
- **ABNT NBR 6122** — Projeto e execução de fundações.
- **ABNT NBR 5674** — Manutenção de edificações — Requisitos para o sistema de gestão de manutenção.
- **ABNT NBR 5626** — Sistemas prediais de água fria e água quente.
- **ABNT NBR 8160** — Sistemas prediais de esgoto sanitário.
- **ABNT NBR 10844** — Instalações prediais de águas pluviais.
- **ABNT NBR 5410** — Instalações elétricas de baixa tensão.
- **ABNT NBR 5419** — Proteção contra descargas atmosféricas, quando aplicável.
- **ABNT NBR 9575** — Impermeabilização — Seleção e projeto.
- **ABNT NBR 9574** — Execução de impermeabilização.
- **ABNT NBR 13753** — Revestimento de piso interno ou externo com placas cerâmicas.
- **ABNT NBR 13754** — Revestimento de paredes internas com placas cerâmicas.
- **ABNT NBR 13755** — Revestimento de paredes externas e fachadas com placas cerâmicas.
- **ABNT NBR 10821** — Esquadrias para edificações.
- **ABNT NBR 13245** — Execução de pinturas em edificações não industriais.

---

## Padrão de aprovação

| Resultado | Ação |
|---|---|
| Aprovado | Serviço liberado para próxima etapa |
| Aprovado com restrição | Pode avançar, mas a pendência precisa ser corrigida e registrada |
| Reprovado | Não libera próxima etapa até correção e nova vistoria |

# FVS-001 — Locação de obra / gabarito

## Objetivo

Verificar se a locação da obra, eixos, alinhamentos, esquadros, níveis e referências foram executados conforme projeto arquitetônico, estrutural e levantamento topográfico.

| 5 | Equipamentos de medição disponíveis e em condição de uso |  |  |  |

## Checklist de execução

| Item | Verificação | C | NC | NA |
|---|---|---|---|---|
| 1 | Gabarito executado fora da área de escavação e concretagem |  |  |  |
| 2 | Pontaletes firmes, alinhados e travados |  |  |  |
| 3 | Tábuas do gabarito niveladas e fixadas corretamente |  |  |  |

| 9 | Locação das fundações compatível com projeto estrutural |  |  |  |
| 10 | Registro fotográfico realizado antes da escavação |  |  |  |

## Critérios de aceitação

A locação deve estar compatível com os projetos liberados. Não deve haver divergência entre eixos, recuos, níveis e posição das fundações. Qualquer incompatibilidade entre arquitetura, estrutura e topografia deve ser tratada antes do início das escavações.

## Registros obrigatórios

Fotos do gabarito, fotos dos eixos principais, croqui de conferência, assinatura do responsável técnico ou encarregado autorizado.
